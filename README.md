# Product Class - Java

## Descripción
Este proyecto demuestra la implementación de una clase en Java utilizando:
- Atributos privados
- Constructores (por defecto y parametrizado)
- Métodos Getter y Setter
- Sobrescritura del método toString()

## Cómo compilar

javac Product.java
javac Main.java

## Cómo ejecutar

java Main

## Ejemplo de salida

Nombre del producto 1: Laptop
Precio del producto 2: 80.0
Producto [ID: P001, Nombre: Laptop, Precio: 2500.0, Stock: 10]
Producto [ID: P002, Nombre: Mouse, Precio: 80.0, Stock: 50]
